# Ledger

## Introduction

A web application to ease the burden of managing financial accounts across multiple institutions.

## Tech Stack
- Google Sheets
- ASP.NET
- React
- Material UI

## Client

### Initialize

- <code>cd client</code>
- <code>npm install</code>

### Development

- <code>cd client</code>
- <code>npm start</code>

### Build

- <code>cd client</code>
- <code>npm run build</code>

## Server

- <code>cd server</code>
- <code>dotnet run</code>

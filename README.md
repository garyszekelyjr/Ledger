# Ledger

## Introduction

A lightweight desktop application to ease the burden of managing financial accounts across multiple institutions.

## Tech Stack
- React
- Python
- Flask
- SQLite

## Instructions

### Client

#### Initial

- <code>cd client</code>
- <code>npm install</code>

#### Development

- <code>cd client</code>
- <code>npm start</code>

#### Build

- <code>cd client</code>
- <code>npm run build</code>

### Server

- <code>cd server</code>
- <code>python3 main.py</code>
